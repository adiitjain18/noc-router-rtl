# Network-on-Chip Router (RTL)

This repository contains a step-by-step RTL implementation of a Network-on-Chip (NoC) router written in Verilog.

## Tools
- ModelSim
- VS Code
- Verilog HDL

## Project Structure

rtl/     → RTL modules  
tb/      → testbenches  
sim/     → simulation workspace  
scripts/ → ModelSim scripts
